状态码

HTTP协议的状态码由3位数字组成，第一个数字定义了响应的类别，共有5中类别：

1.1xx: 指示信息–表示请求已接收，继续处理

2.2xx: 成功–表示请求已被成功接收、理解、接受

3.3xx: 重定向–要完成请求必须进行更进一步的操作

4.4xx: 客户端错误–请求有语法错误或请求无法实现

5.5xx: 服务器端错误–服务器未能实现合法的请求
